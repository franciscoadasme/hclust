# hclust

Fast hierarchical clustering algorithms in pure Crystal.

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     hclust:
       github: franciscoadasme/hclust
   ```

2. Run `shards install`
